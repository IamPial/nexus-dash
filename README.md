##  Overview

NexusDash is a modern full-stack travel and experience booking platform built with **Next.js, TypeScript, Express.js, and MongoDB**.

The platform allows users to discover premium travel destinations, explore curated experiences, search and filter listings, view detailed destination information, and manage travel packages through a secure authentication system.

Designed with scalability, performance, and clean UI/UX principles in mind, NexusDash delivers a seamless booking experience across desktop, tablet, and mobile devices.

#  Installation

Clone the repository

```bash
git clone https://github.com/IamPial/nexus-dash.git
```

Go to project folder

```bash
cd NexusDash
```

Install dependencies

```bash
npm install
```

Run frontend

```bash
npm run dev
```

Run backend

```bash
npm run dev
```
